# Spacing

Utilities for vertical alignment and text-alignment, careful not to be confused with content alignment. This will work on text and images but possibly not on some child elements.

These are all applied as !important due to the nature of their usage.

```
._m(0-5|auto)
._m:x(0-5|auto)
._m:y(0-5|auto)
._m:left(0-5|auto)
._m:right(0-5|auto)
._m:top(0-5|auto)
._m:bottom(0-5|auto)
._p(0-5|auto)
._p:x(0-5|auto)
._p:y(0-5|auto)
._p:left(0-5|auto)
._p:right(0-5|auto)
._p:top(0-5|auto)
._p:bottom(0-5|auto)
```
