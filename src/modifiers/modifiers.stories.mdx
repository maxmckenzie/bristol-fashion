# Modifiers

To avoid calshing with other classes and or frameworks that may be in use in the same project as Bristol fashion is used. The modifiers all have `_` prepended to them.

## Content Layout

Utility CSS classes for flexbox. `._content` include flex property apply ._flex-column to change default row in browser

> Make sure to apply these to the parent container

```
._content-centre
._content-centre-x
._content-centre-y
._content-align-top
._content-align-bottom
._content-align-left
._content-align-right
._content-space-between-x
._content-space-between-y
._content-space-around-x
._content-space-around-y
._content-stretch-x
._content-stretch-y
._flex-column
._flex-row
._flex-column-reverse
._flex-row-reverse
._flex-wrap-reverse
._fill
```

## Alignment

Utilities for vertical alignment and text-alignment, careful not to be confused with content alignment. This will work on text and images but possibly not on some child elements

```
._align-left
._align-right
._align-centre
._align-middle
._align-baseline
._align-top
._align-bottom
._align-sub
._align-text-top
```
