# Alignment

Utilities for vertical alignment and text-alignment, careful not to be confused with content alignment. This will work on text and images but possibly not on some child elements

```
._align:left
._align:right
._align:center
._align:middle
._align:baseline
._align:top
._align:bottom
._align:sub
._align:text-top
```
