## Content Layout

Utility CSS classes for flexbox. `._content` includes flex property appled `._flex-column` or `._flex-row` to change default row in browser. Make sure to apply these to the parent container.

```
._content-center
._content-center:x
._content-center:y
._content-align:top
._content-align:bottom
._content-align:left
._content-align:right
._content-space-between:x
._content-space-between:y
._content-space-around:x
._content-space-around:y
._content-stretch:x
._content-stretch:y
._flex-column
._flex-row
._flex-column-reverse
._flex-row-reverse
._flex-wrap-reverse
```

These are designed to be used as classes in HTML `<div class="_content-center"><!-:your centered content--></div>`

We apply `align-items: baseline;` to avoid the element stretching, If this is what you want you can just apply `align-items: stretch;` that to your element after you have loaded Bristols CSS. Where given a choice bristol will attempt to keep the div the size of the content.
