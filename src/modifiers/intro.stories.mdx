# About Modifiers

Modifiers provide a set of default classes that can be used within html templates. To avoid calshing with other classes and or frameworks that may be in use in the same project as Bristol fashion. The modifiers have `_` prepended to them.

export const parentStyle = {
  backgroundColor: 'rgb(207,216,220)',
  height: '7rem',
  padding: '0.5rem',
  margin: '0 0 1rem'
}

export const childStyle = {
  backgroundColor: 'rgb(55,71,79)',
  margin: '0.25rem',
  padding: '0.5rem',
  color: 'rgb(207,216,220)',
  borderRadius: '0.25rem',
  fontFamily: '"Noto serif", sans-serif',
  lineHeight: '1',
}

import './content.less'

<Story name="Default">
  {() => {
    return (
      <div>
        <div style={parentStyle} class="_content-center">
          <div style={childStyle}>_content-center</div>
          <div style={childStyle}>_content-center</div>
        </div>
      </div>
    )
  }}
</Story>

These can then be used in html

```
<div class="_content-center">
  <!--content-->
</div>
```

Or if you perfer you can use them as a mixin in Less

```
.element {
  ._content-center;
}
```

With Sass/Scss you unfortunatly cannot pull in classes as mixins like this.
