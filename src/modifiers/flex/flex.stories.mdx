## Flex layout modifiers

Utility CSS classes for flex box. `._flex` includes flex property applied `._flex:column` or `._flex:row` to change default row in browser. Make sure to apply these to the parent container.

```css
._flex
._flex:column
._flex:column-reverse
._flex:row
._flex:row-reverse
._flex:nowrap
._flex:wrap
._flex:wrap-reverse
._flex:center
._flex:justify(start)
._flex:justify(end)
._flex:justify(center)
._flex:justify(space-around)
._flex:justify(space-between)
._flex:align(stretch)
._flex:align(start)
._flex:align(end)
._flex:align(center)
._flex:wrap(stretch)
._flex:wrap(start)
._flex:wrap(end)
._flex:wrap(center)
._flex:wrap(space-around)
._flex:wrap(space-between)
```

These are designed to be used as classes in HTML `<div class="_flex:column"><!--content--></div>`
