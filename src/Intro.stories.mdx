Bristol fashion is a collection of design system utilities, CSS variables and defaults; rather than providing developers with a UI framework such as Bootstrap and Foundation. Bristol provides the utilities you need to roll your own based around core design theory and a little bit of maths.

Source: [Github source](https://github.com/maxmckenzie/bristol-fashion) you can also see the planned features and project management here [project board](https://github.com/maxmckenzie/bristol-fashion/projects)

## Install

You can include just the CDN if you would like to https://bristol-fashion.now.sh/styles.css Including this in your head will include all of the CSS and all of the browser defaults that come with it.

```
<link rel="stylesheet" href="https://bristol-fashion.now.sh/styles.css">
```

or `yarn add bristol-fashion` or `npm i bristol-fashion` depending on your preference.

## Features

- normalize.css
- Typography utility for type scale calculation and browser defaults
- Color CSS variables are broken down into Brand, Pallet, neutral, and UI
- Inset and drop shadow defaults
- Modifier classes with `._`
