# About Variables

We all know and love CSS3 variables, They can be loaded once and used across the website. Esspecially handy if your using a framework like react or vue. This library provides a set of common defaults in a sensible namespace.

<img src={variables} width="400px" />

For colors `--brand--{type}`, `--ui-{feedback}-{tone}`, and `--neutral-{tone}`, and For shadows `--shadow-{depth}` i.e `--shadow-1` `--shadow-2` and used as inset and drop shadow

```
box-shadow: var(--shadow-1);
box-shadow: inset var(--shadow-1);
``` 

These can be overwritten and extended as you like
