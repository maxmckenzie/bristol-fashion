# Usage

As well as being included in the minified full bristol.css file you can import just the variables, functions and or browser default. 

> Just including the CSS from the cdn this provides you a set of defaults for h1, h2, h3 etc. there are also two additional classes of `.textSmall` `.textTiny`.

Browser defaults (include variables and functions)

built CSS
```
@import "bristol-fashion/src/typography/typography.css";
```

or you can include the utilities and overwrite the variables with your own

yourFile.less
```
:root {
  font-size: 16px;
}

@base-font-size: 1rem;
@base-font-ratio: 1.25; // major second
@type-margin: 0;

@import "bristol-fashion/src/typography/typography-functions.less";
@import "bristol-fashion/src/typography/typography.less";

```

yourFile.scss
```
:root {
  font-size: 16px;
}

@base-font-size: 1rem;
@base-font-ratio: 1.25; // major second
@type-margin: 0;

@import "bristol-fashion/src/typography/typography.scss";

```
