# Shadows

Shadows are available as CSS Variables. This is handy if you want to set an inset shadow as per the below example

```
box-shadow: inset var(--shadow-1);
```

then for a drop shadow

```
box-shadow: var(--shadow-1);
```

the variables available are

```
--shadow-1
--shadow-2
--shadow-3
--shadow-4
--shadow-5
```
