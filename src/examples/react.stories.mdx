# Example using React

Install 

```
npm i bristol-fashion
```

Then import the css file into your app.js/jsx

CSS

```
import React, { Suspense, lazy } from 'react';
import ReactDOM from 'react-dom';

import App from './App'

import 'bristol-fashion/src/index.css'

```

Less

```
import React, { Suspense, lazy } from 'react';
import ReactDOM from 'react-dom';

import App from './App'

import 'bristol-fashion/src/index.less'

ReactDOM.render(
  <App />,
  document.getElementById('root'),
);


```

Sass/Scss
```
import React, { Suspense, lazy } from 'react';
import ReactDOM from 'react-dom';

import App from './App'

import 'bristol-fashion/src/index.scss'

ReactDOM.render(
  <App />,
  document.getElementById('root'),
);

```

You can also include just the type scale functions to use in your app

```
import React, { Suspense, lazy } from 'react';
import ReactDOM from 'react-dom';

import App from './App'

import "bristol-fashion/src/typography/typography-functions.less";

```
