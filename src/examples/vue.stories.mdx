# Example using Vue

install 

```
npm i bristol-fashion
```

then import the css file from your app.js/jsx

CSS

```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.css'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');

```

Less

```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.less'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');


```

Sass/Scss
```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.scss'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');
```

You can also include jsut the type scale function is you want as well

```
import Vue from 'vue';
import App from './App'

import "bristol-fashion/src/typography/typography-functions.less";
```
