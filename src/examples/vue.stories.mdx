# Example using Vue

Install 

```
npm i bristol-fashion
```

Then import the css file into your app.js/jsx

CSS

```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.css'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');

```

Less

```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.less'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');


```

Sass/Scss
```
import Vue from 'vue';
import App from './App'

import 'bristol-fashion/src/index.scss'

const app = new Vue({
  render: h => h(App)
}).$mount('#app');
```

You can also include just the type scale functions to use in your app

```
import Vue from 'vue';
import App from './App'

import "bristol-fashion/src/typography/typography-functions.less";

```
