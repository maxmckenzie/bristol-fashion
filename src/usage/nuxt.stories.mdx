# Example using Nuxt

Install 

```
npm i bristol-fashion
```

Then import the css file into your nuxt.config.js

CSS

```
/*
  ** Global CSS
  */
  css: [
    'bristol-fashion/src/index.css'
  ],

```

the same can be done for less and scss
